windowsとは？

Windowsとは、米マイクロソフト（Microsoft）社が開発・販売している、 コンピュータのオペレーティングシステム（OS）製品のシリーズ名。 単にWindowsといった場合にはパソコン向けの製品系列を指すことが多いが、 他にサーバ向けや携帯機器向けなども展開している。

windowsOSの歴史年表

  • 1975年

    ビル・ゲイツとポール・アレンによってMicrosoft社を設立

  • 1985年

    windows1.0が発売

    第１弾となって性能はあまり良くなく、評判はイマイチだった

    日本語版が発売されたのは1987年

  • 1987年

    windows2.0が発売

    画面がタイル型からオーバーラップ型になり、 複数のウィンドウを重ねて表示することができるようになった

    windows1.0の課題であった動作の重さが解消され、処理が高速化

    日本語版は1988年に発売

  • 1990年

    windows3.0が発売

    下記のような追加機能により、以前よりも実用的なwindowsOSになった

    ・複数のMS-DOSアプリケーションをマルチタスク化

    ※MS-DOS(MicroSoft Disk Operating System（マイクロソフト・ディスク・オペレーティング・システム）k)とはMIcrosoftが作ったOS。

    ・ネットワーク機能

    日本語版は1991年に発売

  • 1992年

    windows3.1が発売

    マルチメディア(動画や音声)機能が搭載

    安定性が向上し、より実用性が高まった

    windows NT3.1~4.0

    1994年に「windows NT 3.1」、1995年に「windows NT 3.5」、1996年に「windows NT 4.0」が発売

    NTシリーズは当初、企業向け製品に搭載されていたものだが、NT系の高い安定性が評価されたことで、 一般向け製品にもなった

  • 1995年

    windows 95が発売

    Startボタン、メニュー、タスクバー、ウィンドウの最大・最小化などの今でも日常的に使っている様々な便利機能が搭載

    機能性・実用性を向上させたWindows 95は、発売4日で400万本を売り上げを記録。これ以降に開発されるOSの標準としての地位を獲得

  • 1998年

    windows 98が発売

    Windows 95をさらに個人消費者向けに改良したもの

    全体的なパフォーマンス向上のほか、DirectX5.0搭載、USB・DVD・マルチモニターへの対応などが特徴。 安定したOSとして高い売上を記録

  • 2000年

    windows2000が発売

    安定性・堅牢性に優れたWindows NT 4.0を基に構築されたOS

    その一方で、あまり個人消費者向けではないと判断された

  • 2000年

    windows MEが発売

    「Millennium Edition」の略。Windows 2000が業務用OSとして位置づけされたことにより、急遽リリースされたO

    短い開発期間の中で多機能性を追求した結果、バグが多くクラッシュも頻繁に発生。そのため評価はイマイチ

  • 2001年

    windows XPが発売

    多くのユーザーから長く愛されるロングセラーとなったOS

    NT系の安定性と堅牢性、9x系の多機能性と操作性が1つに統合され、見た目のデザインを一新

  • 2007年

    windows Vistaが発売

    多くの新機能が搭載され、画面のデザインも大幅に変更

    これまでのWindowsとは違った印象をユーザーに与えることができた一方、操作手順も変わってしまったことで「使いづらい」という評価を受けた

  • 2009年

    windows7が発売

    windows Vistaの後継版で、高速起動、マルチタスク処理の高速化、操作性、グラフィックス、アプリケーションなど様々な改良された

    サポートは既に終了しているが、今現在でも使っている人は少数いる

  • 2012年

    windows8が発売

    タブレット端末を意識し、タッチスクリーンデバイスへの対応に力を入れた

    しかし、スタートボタンの廃止など斬新すぎるデザインに「前の方が良かった」という声も多数上がった

    ちなみにスタートボタンやスタート画面は、その後アップデート版として開発されたWindows 8.1で復活している

  • 2015年

    windows10が発売

    Windows 8で一度削除されたスタート メニューが復活。音声認識アシスタント機能「Cortana」などの先進的な機能が搭載

    ローリングリリースモデルが採用され、今後は機能改善や新機能追加などのアップデータはWindows Updateを通じて無償で行われる

    Windows 10が継続して更新されていくことになるため、従来のようなメジャーアップデートはなくなった

  • 2021年

    windows11が発売

    UIが変更され、全体的に丸みを帯びたガラススタイルのデザインに。またスタート メニューが再設計され、アイコンボタンのみを設置する形になっている

    ウィンドウを整列させる「スナップ機能」や「ウィジェット機能」により、ユーザーが必要とする情報を画面上に表示できるようになるなど、さまざまな機能が追加
